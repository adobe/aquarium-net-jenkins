Optional regular expression pattern to filter which labels this cloud will serve.

If specified, only Fish labels whose names match this pattern will be available for provisioning. If left empty, all labels from the Fish cluster will be available.

Example patterns:

  • ubuntu.* - Only labels starting with "ubuntu"
  • .*gpu.* - Only labels containing "gpu"
  • linux-(16|18|20) - Labels matching linux-16, linux-18, or linux-20
