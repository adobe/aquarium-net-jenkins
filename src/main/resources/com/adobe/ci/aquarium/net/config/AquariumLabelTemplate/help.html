Configure Aquarium Label Templates for use in pipeline steps.

Template Content should be valid YAML that defines an Aquarium Label. You can use variables like ${variableName} that will be substituted when the template is used.

Predefined Variables:

  • ${TIMESTAMP} - Current timestamp in format YYMMDD.hhmmss
  • ${NOW+N*UNIT} - Current time plus offset (e.g., ${NOW+3*HOUR})

Example Template:

name: temp_ci-${name}-${TIMESTAMP}
version: 0
visible_for:
  - jenkins
  - jenkins-group
remove_at: '${NOW+3*HOUR}'
definitions:
  - driver: aws
    images:
      - name: '${image}'
    options:
      instance_type: g4ad.8xlarge
      security_groups:
        - supergroup1
        - supergroup2
      userdata_format: env
    resources:
      cpu: 32
      ram: 128
      network: Subnet:build
      lifetime: 4h
metadata:
  JENKINS_AGENT_WORKSPACE: /mnt/ws
