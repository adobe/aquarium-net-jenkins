Creates an Aquarium Label from a predefined template. Select a template and provide values for any variables defined in the template.

Predefined Variables:
  • ${TIMESTAMP} - Current timestamp in format YYMMDD.hhmmss
  • ${NOW+N*UNIT} - Current time plus offset (e.g., ${NOW+3*HOUR} for 3 hours from now)
Supported time units: SECOND, MINUTE, HOUR, DAY, WEEK, MONTH, YEAR
